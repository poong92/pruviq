# docs/archive/ — 시점 스냅샷 보관소

이 디렉토리는 **과거 특정 시점의 기록물**을 담습니다. 현재 운영에 필요한
리빙 문서는 `docs/` 최상위에 있고, 이 아래로 떨어진 파일들은:

- 작성 시점의 상태 기록 (예: QA 감사, 성능 리포트)
- 버전 이전 로드맵 (예: v1.1 — 현행은 v1.2)
- 1회성 리서치 스냅샷 (예: 법률 환경 조사)

## 카테고리

| 디렉토리 | 용도 |
|----------|------|
| `qa/` | QA 감사 리포트, UX 감사, 시뮬레이터 버그픽스 감사 |
| `audits/` | 경쟁사/통합 감사 (v0.1.0 등) |
| `roadmap/` | 이전 버전 로드맵 (v1.1 등) |
| `legal/` | 법률 리서치 시점 스냅샷 |
| `research/` | 엔진 패리티 등 1회성 리서치 |

## 규칙

- **편집 금지 원칙**: 스냅샷은 당시 상태 보존용. 오타 교정도 지양.
- **이 디렉토리 내부 참조는 `docs-lint` 가드 제외** — 과거 OpenClaw/Mac Mini
  언급이 남아있어도 CI red 아님.
- 현행 상태 참조가 필요하면 `docs/` 최상위 파일 (ARCHITECTURE.md,
  INFRASTRUCTURE.md, QA_AUTOMATION.md, DOC_INDEX.md 등) 이용.

## 아카이브 기준

60일 이상 갱신 없음 + 리빙 문서 대체본 존재 → archive 대상.
PR5 도입 예정인 `docs-staleness.yml` 이 이 기준으로 자동 warn/error 발생.
